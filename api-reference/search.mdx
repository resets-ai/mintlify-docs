# Search API Reference

## Base URL

```
https://api.resets.ai
```

## Authentication

All API requests require an API key in the header:

<CodeGroup>

```bash curl
--header "x-api-key: rst_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"
```


```javascript javascript
headers: {
  'X-API-Key': 'rst_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx'
}
```


```python python
headers = {
    "X-API-Key": "rst_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"
}
```

</CodeGroup>

## Search

<Card title="POST /api/search" icon="magnifying-glass">
  Search across multiple engines with optional AI enhancement
</Card>

### Authorization

<ParamField header="X-API-Key" type="string" required>
  Your API key in the format `rst_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`
</ParamField>

### Request Body

<ParamField body="query" type="string" required>
  Search query with support for boolean operators. Length: 1-1000 characters.

  Examples:

  - Simple: `"artificial intelligence"`
  - Boolean: `"(AI OR ML) AND healthcare"`
  - Site-specific: `"site:linkedin.com/company tesla"`
</ParamField>

<ParamField body="limitResults" default="10" type="number">
  Number of results to return. Range: 1-500.

  Credit cost: 5 credits per 100 results

  - 10 results = 5 credits
  - 100 results = 5 credits
  - 500 results = 25 credits
</ParamField>

<ParamField body="date" type="string">
  Filter results by recency. Options:

  - `h` - Past hour
  - `d` - Past 24 hours
  - `w` - Past week
  - `m` - Past month
  - `y` - Past year
</ParamField>

<ParamField body="countryCode" type="string">
  Two-letter country code for localization.

  Examples: `us`, `uk`, `de`, `fr`, `jp`
</ParamField>

<ParamField body="language" type="string">
  Two letter anguage code for content filtering.

  Examples: `en`, `es`, `de`, `fr`, `zh`
</ParamField>

<ParamField body="pageSummary" type="object">
  Configuration for AI-powered data extraction.

  <Expandable title="pageSummary properties">
    <ParamField body="prompt" type="string">
      Natural language extraction instructions. Max 2500 characters.
    </ParamField>
    <ParamField body="schema" type="object">
      JSON schema for structured output. Max 2500 characters.

      If only schema is provided, a default prompt is auto-generated.
    </ParamField>
  </Expandable>
</ParamField>

### Response

<ResponseField name="success" type="boolean" required>
  Whether the request was successful
</ResponseField>

<ResponseField name="data" type="object" required>
  <Expandable title="data properties">
    <ResponseField name="requestId" type="string" required>
      Unique identifier for the request (UUID v4)
    </ResponseField>
    <ResponseField name="results" type="array" required>
      Array of search results

      <Expandable title="result properties">
        <ResponseField name="title" type="string" required>
          Page title or "Untitled" if not available
        </ResponseField>
        <ResponseField name="url" type="string" required>
          Complete URL including protocol
        </ResponseField>
        <ResponseField name="snippet" type="string" required>
          Text excerpt, typically 150-300 characters
        </ResponseField>
        <ResponseField name="position" type="number" required>
          Sequential position after deduplication (1-based)
        </ResponseField>
        <ResponseField name="dateTime" type="string">
          ISO 8601 publication timestamp when available
        </ResponseField>
        <ResponseField name="dateReference" type="string">
          Human-friendly date like "2 days ago"
        </ResponseField>
        <ResponseField name="pageSummary" type="object">
          AI-extracted structured data (only when pageSummary requested)
        </ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="metadata" type="object" required>
      <Expandable title="metadata properties">
        <ResponseField name="costCredits" type="number" required>
          Total credits consumed for this request
        </ResponseField>
        <ResponseField name="responseTime" type="number" required>
          Processing duration in seconds (3 decimal places)
        </ResponseField>
        <ResponseField name="pageSummaryProcessed" type="boolean" required>
          Whether AI enhancement was applied
        </ResponseField>
        <ResponseField name="totalResults" type="number" required>
          Final count after deduplication
        </ResponseField>
        <ResponseField name="resultsRequested" type="number" required>
          Original limitResults parameter value
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash curl
curl -X POST https://api.resets.ai/api/search \
  --header "content-type: application/json" \
  --header "x-api-key: YOUR_API_KEY" \
  --data '{
    "query": "AI startups series B funding",
    "limitResults": 20,
    "date": "m",
    "pageSummary": {
      "schema": {
        "company": "string",
        "amount": "string",
        "investors": ["string"]
      }
    }
  }'
```


```javascript javascript
const response = await fetch('https://api.resets.ai/api/search', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': 'YOUR_API_KEY'
  },
  body: JSON.stringify({
    query: "AI startups series B funding",
    limitResults: 20,
    date: "m",
    pageSummary: {
      schema: {
        company: "string",
        amount: "string",
        investors: ["string"]
      }
    }
  })
});
const data = await response.json();
```


```python python
import requests
import json

url = "https://api.resets.ai/api/search"
headers = {
    "Content-Type": "application/json",
    "X-API-Key": "YOUR_API_KEY"
}
data = {
    "query": "AI startups series B funding",
    "limitResults": 20,
    "date": "m",
    "pageSummary": {
        "schema": {
            "company": "string",
            "amount": "string",
            "investors": ["string"]
        }
    }
}
response = requests.post(url, headers=headers, json=data)
result = response.json()
```

## Rate Limits

API rate limits vary by plan:

| Plan       | Requests/Second | Burst Capacity |
| ---------- | --------------- | -------------- |
| Free       | 5               | 10             |
| Enterprise | Custom          | Custom         |

### Rate Limit Headers

```
X-RateLimit-Limit: 5
X-RateLimit-Remaining: 4
X-RateLimit-Reset: 1640995200
```

### Rate Limit Error

```json
{
  "success": false,
  "error": {
    "httpStatusCode": 429,
    "code": "RATE_LIMIT_EXCEEDED",
    "message": "Rate limit exceeded for your plan",
    "details": {
      "retryAfter": 60
    }
  }
}
```

## Error Codes

All errors follow this format:

```json
{
  "success": false,
  "error": {
    "httpStatusCode": 400,
    "code": "ERROR_CODE",
    "message": "Human-readable error message",
    "details": {}
  }
}
```

### Common Error Codes

| Code                    | HTTP Status | Description               |
| ----------------------- | ----------- | ------------------------- |
| `MISSING_API_KEY`       | 401         | API key not provided      |
| `INVALID_API_KEY`       | 401         | API key is invalid        |
| `INSUFFICIENT_CREDITS`  | 402         | Not enough credits        |
| `RATE_LIMIT_EXCEEDED`   | 429         | Too many requests         |
| `INVALID_URL_FORMAT`    | 400         | URL is malformed          |
| `EXTRACTION_FAILED`     | 400         | Could not extract content |
| `QUERY_TOO_LONG`        | 400         | Query exceeds 1000 chars  |
| `INTERNAL_SERVER_ERROR` | 500         | Server error              |

| Header                | Description                     |

| `X-Credits-Remaining` | Your remaining credit balance   |
