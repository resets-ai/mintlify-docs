# Extract API Reference

## Base URL

```
https://api.resets.ai
```

## Authentication

All API requests require an API key in the header:

<CodeGroup>

```bash curl
--header "x-api-key: rst_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"
```


```javascript javascript
headers: {
  'X-API-Key': 'rst_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx'
}
```


```python python
headers = {
    "X-API-Key": "rst_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"
}
```

</CodeGroup>

## Extract

<Card title="POST /api/extract" icon="globe">
  Extract content from URLs with intelligent routing and optional AI processing
</Card>

### Authorization

<ParamField header="X-API-Key" type="string" required>
  Your API key in the format `rst_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`
</ParamField>

### Request Body

<ParamField body="url" type="string" required>
  Target URL for extraction. Must include protocol (http:// or https://).

  Valid: `https://example.com`\
  Invalid: `example.com`, `www.example.com`
</ParamField>

<ParamField body="pageSummary" type="object">
  Configuration for AI-powered data extraction.

  <Expandable title="pageSummary properties">
    <ParamField body="prompt" type="string">
      Natural language extraction instructions. Max 2500 characters.
    </ParamField>
    <ParamField body="schema" type="object">
      JSON schema for structured output. Max 2500 characters.

      If only schema is provided, a default prompt is auto-generated.
    </ParamField>
  </Expandable>
</ParamField>

### Response

<ResponseField name="success" type="boolean" required>
  Whether the request was successful
</ResponseField>

<ResponseField name="data" type="object" required>
  <Expandable title="data properties">
    <ResponseField name="requestId" type="string" required>
      Unique identifier for the request (UUID v4)
    </ResponseField>
    <ResponseField name="result" type="object" required>
      <Expandable title="result properties">
        <ResponseField name="url" type="string" required>
          Original requested URL
        </ResponseField>
        <ResponseField name="title" type="string" required>
          Extracted page title
        </ResponseField>
        <ResponseField name="content" type="string" required>
          Full text content (truncated to 200,000 characters)
        </ResponseField>
        <ResponseField name="extractedAt" type="string" required>
          ISO 8601 timestamp of extraction
        </ResponseField>
        <ResponseField name="processingTime" type="number" required>
          Extraction duration in seconds
        </ResponseField>
        <ResponseField name="pageSummary" type="object">
          AI-processed structured data (only when pageSummary requested)
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="metadata" type="object" required>
  <Expandable title="metadata properties">
    <ResponseField name="costCredits" type="number" required>
      Credits consumed (1 for basic, 2 with AI)
    </ResponseField>
    <ResponseField name="responseTime" type="number" required>
      Total request duration in seconds
    </ResponseField>
    <ResponseField name="domain" type="string" required>
      Base domain (e.g., "example.com")
    </ResponseField>
    <ResponseField name="httpStatusCode" type="number" required>
      HTTP status from extraction
    </ResponseField>
    <ResponseField name="pageSummaryProcessed" type="boolean" required>
      Whether AI processing was applied
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash curl
curl -X POST https://api.resets.ai/api/extract \
  --header "content-type: application/json" \
  --header "x-api-key: YOUR_API_KEY" \
  --data '{
    "url": "https://www.ycombinator.com/companies/stripe",
    "pageSummary": {
      "prompt": "Extract company details",
      "schema": {
        "name": "string",
        "founded": "string",
        "founders": ["string"],
        "description": "string",
        "funding": "string"
      }
    }
  }'
```


```javascript javascript
const response = await fetch('https://api.resets.ai/api/extract', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': 'YOUR_API_KEY'
  },
  body: JSON.stringify({
    url: "https://www.ycombinator.com/companies/stripe",
    pageSummary: {
      prompt: "Extract company details",
      schema: {
        name: "string",
        founded: "string",
        founders: ["string"],
        description: "string",
        funding: "string"
      }
    }
  })
});
const data = await response.json();
```


```python python
import requests
import json

url = "https://api.resets.ai/api/extract"
headers = {
    "Content-Type": "application/json",
    "X-API-Key": "YOUR_API_KEY"
}
data = {
    "url": "https://www.ycombinator.com/companies/stripe",
    "pageSummary": {
        "prompt": "Extract company details",
        "schema": {
            "name": "string",
            "founded": "string",
            "founders": ["string"],
            "description": "string",
            "funding": "string"
        }
    }
}
response = requests.post(url, headers=headers, json=data)
result = response.json()
```

## Rate Limits

API rate limits vary by plan:

| Plan       | Requests/Second | Burst Capacity |
| ---------- | --------------- | -------------- |
| Free       | 5               | 10             |
| Enterprise | Custom          | Custom         |

### Rate Limit Headers

```
X-RateLimit-Limit: 5
X-RateLimit-Remaining: 4
X-RateLimit-Reset: 1640995200
```

### Rate Limit Error

```json
{
  "success": false,
  "error": {
    "httpStatusCode": 429,
    "code": "RATE_LIMIT_EXCEEDED",
    "message": "Rate limit exceeded for your plan",
    "details": {
      "retryAfter": 60
    }
  }
}
```

## Error Codes

All errors follow this format:

```json
{
  "success": false,
  "error": {
    "httpStatusCode": 400,
    "code": "ERROR_CODE",
    "message": "Human-readable error message",
    "details": {}
  }
}
```

### Common Error Codes

| Code                    | HTTP Status | Description               |
| ----------------------- | ----------- | ------------------------- |
| `MISSING_API_KEY`       | 401         | API key not provided      |
| `INVALID_API_KEY`       | 401         | API key is invalid        |
| `INSUFFICIENT_CREDITS`  | 402         | Not enough credits        |
| `RATE_LIMIT_EXCEEDED`   | 429         | Too many requests         |
| `INVALID_URL_FORMAT`    | 400         | URL is malformed          |
| `EXTRACTION_FAILED`     | 400         | Could not extract content |
| `QUERY_TOO_LONG`        | 400         | Query exceeds 1000 chars  |
| `INTERNAL_SERVER_ERROR` | 500         | Server error              |

All responses include these headers:

| Header                | Description                     |
| --------------------- | ------------------------------- |
| `X-Request-ID`        | Unique request identifier       |
| `X-Response-Time`     | Processing time in milliseconds |
| `X-Credits-Remaining` | Your remaining credit balance   |
