## Step 3: Make Your First Search Request in Playground

Let's search for "YC or Y Combinator companies in AI or Articifical Intelligence" across all search engines:

<CodeGroup>

```bash curl
curl -X POST https://api.resets.ai/api/search \
  --header "content-type: application/json" \
  --header "x-api-key: YOUR_API_KEY" \
  --data '{
    "query": "site:www.linkedin.com/company (yc OR y combinator) AND ( ai OR artifical intelligence)",
    "limitResults": 10
}'
```


```javascript javascript
const response = await fetch('https://api.resets.ai/api/search', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': 'YOUR_API_KEY'
  },
  body: JSON.stringify({
    query: "AI companies",
    limitResults: 10
  })
});
const data = await response.json();
console.log(data);
```


```python python
import requests
import json

url = "https://api.resets.ai/api/search"
headers = {
    "Content-Type": "application/json",
    "X-API-Key": "YOUR_API_KEY"
}
data = {
    "query": "AI companies",
    "limitResults": 10
}

response = requests.post(url, headers=headers, json=data)
result = response.json()
print(result)
```

</CodeGroup>

## Step 4: Make Your First Extract Request

Now let's extract content from a specific webpage:

<CodeGroup>

```bash curl
  curl -X POST https://api.yourdomain.com/api/extract \
  -H "Content-Type: application/json" \
  -H "X-API-Key: YOUR_API_KEY" \
  -d '{
    "url": "https://openai.com/about"
  }'
```


```python python
import requests

response = requests.post(
    "https://api.yourdomain.com/api/extract",
    headers={
        "Content-Type": "application/json",
        "X-API-Key": "YOUR_API_KEY"
    },
    json={
        "url": "https://openai.com/about"
    }
)

print(response.json())
```


```javascript javascript
const response = await fetch('https://api.yourdomain.com/api/extract', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': 'YOUR_API_KEY'
  },
  body: JSON.stringify({
    url: 'https://openai.com/about'
  })
});

const data = await response.json();
console.log(data);
```

</CodeGroup>

Common Response Codes

| Status | Description                         |
| ------ | ----------------------------------- |
| 200    | Success - Request completed         |
| 400    | Bad Request - Check your parameters |
| 401    | Unauthorized - Invalid API key      |
| 402    | Insufficient Credits                |
| 429    | Rate Limited - Too many requests    |
| 500    | Server Error - Try again later      |
