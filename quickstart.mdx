# Quick Start Guide

Enter you email and make your first request in playground in secdons. No credit card required.

## Step 1: Get Your Free Credits

<Steps>
  <Step title="Visit the Playground">
    Go to [resets.ai](https://playground.resets.ai) or click the Playground link in the navigation.
  </Step>
  <Step title="Enter Your Email">
    You'll see a simple email input form. Enter your email address and click "Get Started".
  </Step>
  <Step title="Receive 250 Credits">
    You'll instantly receive 250 free credits to explore both search and extraction features.
  </Step>
</Steps>

<Info>
  **No signup required\!** Just enter your email and start using the platform immediately.
</Info>

## Step 2: Get Your API Key

Once you're in the playground:

1. Look for the **"Get API Key"** button in the top navigation
2. Click it to generate your unique API key
3. Copy and save your key securely - it looks like: `rst_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`

<Warning>
  Keep your API key secure and never share it publicly. You can regenerate it anytime from the playground.
</Warning>

## Step 3: Make Your First Search Request in Playground

Let's search for "YC or Y Combinator companies in AI" across all search engines:

<CodeGroup>

```bash
curl -X POST https://api.resets.ai/api/search \
  --header "content-type: application/json" \
  --header "x-api-key: YOUR_API_KEY" \
  --data '{
    "query": "AI companies",
    "limitResults": 10
}'
```


```javascriptconst
response = await fetch('https://api.resets.ai/api/search', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': 'YOUR_API_KEY'
  },
  body: JSON.stringify({
    query: "AI companies",
    limitResults: 10
  })
});
const data = await response.json();
console.log(data);
```


```python
import requests
import json

url = "https://api.resets.ai/api/search"
headers = {
    "Content-Type": "application/json",
    "X-API-Key": "YOUR_API_KEY"
}
data = {
    "query": "AI companies",
    "limitResults": 10
}

response = requests.post(url, headers=headers, json=data)
result = response.json()
print(result)
```

Step 4: Make Your First Extract Request
Now let's extract content from a specific webpage:

<CodeGroup>

```bash
  curl -X POST https://api.yourdomain.com/api/extract \
  -H "Content-Type: application/json" \
  -H "X-API-Key: YOUR_API_KEY" \
  -d '{
    "url": "https://openai.com/about"
  }'
```


```python
import requests

response = requests.post(
    "https://api.yourdomain.com/api/extract",
    headers={
        "Content-Type": "application/json",
        "X-API-Key": "YOUR_API_KEY"
    },
    json={
        "url": "https://openai.com/about"
    }
)

print(response.json())
```


```javascript
const response = await fetch('https://api.yourdomain.com/api/extract', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': 'YOUR_API_KEY'
  },
  body: JSON.stringify({
    url: 'https://openai.com/about'
  })
});

const data = await response.json();
console.log(data);
```
